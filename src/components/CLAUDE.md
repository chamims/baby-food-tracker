# src/components/

All UI components live here, grouped by feature. Never group by file type (e.g. no `modals/` or `buttons/` directories).

| Folder | Purpose |
|--------|---------|
| `Calendar/` | Monthly calendar grid and day tiles |
| `FoodEntry/` | Add-entry modal and expandable entry card |
| `FoodHistory/` | Searchable history list and stats dashboard |
| `Layout/` | App-wide header and bottom navigation bar |
